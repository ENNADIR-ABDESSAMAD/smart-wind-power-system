# ESP libraries (install via Library Manager)

- Blynk (Blynk IoT library)
- ArduinoJson
- SoftwareSerial (built-in on ESP8266)

# Boards

- ESP8266 NodeMCU or ESP32 (update `#include` for ESP32: use `BlynkSimpleEsp32.h`)

# Configuration

Edit `WIFI_SSID`, `WIFI_PASS`, `BLYNK_AUTH_TOKEN`, and `API_URL` in `BlynkBridge.ino` before flashing.
